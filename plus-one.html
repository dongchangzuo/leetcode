Plus One

Problem Description

You are given a large integer represented as an integer array digits, where each digits[i] is the ith digit of the integer.

The digits are ordered from most significant to least significant in left-to-right order.

The large integer does not contain any leading 0's.

Increment the large integer by one and return the resulting array of digits.

Examples

Example 1:

Input: digits = [1,2,3]
Output: [1,2,4]
Explanation: The array represents the integer 123.
Incrementing by one gives 123 + 1 = 124.
Thus, the result should be [1,2,4].

Example 2:

Input: digits = [4,3,2,1]
Output: [4,3,2,2]
Explanation: The array represents the integer 4321.
Incrementing by one gives 4321 + 1 = 4322.
Thus, the result should be [4,3,2,2].

Example 3:

Input: digits = [9]
Output: [1,0]
Explanation: The array represents the integer 9.
Incrementing by one gives 9 + 1 = 10.
Thus, the result should be [1,0].

Constraints

  • 1 ≤ digits.length ≤ 100
  • 0 ≤ digits[i] ≤ 9
  • digits does not contain any leading 0's
1.5s

Algorithm Steps

1. Basic Approach

• Start from the last digit

• Add 1 to the current digit

• If the result is less than 10, we're done

• If the result is 10, set current digit to 0 and carry 1 to the next digit

2. Edge Cases

• If all digits are 9, we need to add a new digit at the beginning

• Example: [9,9,9] becomes [1,0,0,0]

3. Time and Space Complexity

• Time Complexity: O(n) - worst case when we need to add a new digit

• Space Complexity: O(n) - worst case when we need to create a new array

Java
public class Solution {
public int[] plusOne(int[] digits) {
int n = digits.length;
// Start from the last digit
for (int i = n - 1; i >= 0; i--) {
// If current digit is less than 9, just increment and return
if (digits[i] < 9) {
digits[i]++;
return digits;
}
// If current digit is 9, set it to 0
digits[i] = 0;
}
// If we reach here, all digits were 9
int[] newDigits = new int[n + 1];
newDigits[0] = 1;
return newDigits;
}
}

class Solution:
def plusOne(self, digits: List[int]) -> List[int]:
n = len(digits)
# Start from the last digit
for i in range(n - 1, -1, -1):
# If current digit is less than 9, just increment and return
if digits[i] < 9:
digits[i] += 1
return digits
# If current digit is 9, set it to 0
digits[i] = 0
# If we reach here, all digits were 9
